# 金融看板 (Finance Dashboard)

实时市场数据监控看板

## 功能

- 📈 A股指数：上证指数、中证A500
- 🏦 港股：恒生科技、小米集团、长江电力
- 🇺🇸 美股：标普500、纳斯达克、英伟达、谷歌
- 🥇 贵金属：黄金现货
- ₿ 加密货币：比特币、以太坊

## 在线访问

http://43.156.96.119:3000

## 本地运行

```bash
npm install
node server-tencent-finance.js
```

访问 http://localhost:3000

## 技术栈

- Node.js + Express
- 腾讯财经 API
- CoinGecko API (加密货币)
